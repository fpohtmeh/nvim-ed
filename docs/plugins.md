# Plugin Overview

## Core Framework

- **Lazy.nvim**: Plugin manager with lazy loading
- **Snacks.nvim**: UI framework (dashboard, picker, terminal, notifications)

## Development

- **Blink.cmp**: Fast completion engine
- **Mason**: Tool installer (LSP servers, formatters, linters)
- **Treesitter**: Syntax highlighting and parsing
- **LSP**: Language server integration with nvim-lspconfig
- **Overseer**: Task runner and automation
- **Conform**: Auto-format on save

## AI

- **Claude**: AI pair programming via terminal integration (`<leader>a*`)

## File Management

- **Oil.nvim**: File explorer with vim-like navigation
- **Harpoon**: File bookmarking
- **Zoxide**: Smart directory jumping

## Git Integration

- **Fugitive**: Full git workflow
- **GitSigns**: Git decorations and hunk operations
- **Lazygit**: Terminal git interface

## Navigation & Search

- **Flash**: Jump to any location with labels
- **Grug-far**: Advanced search and replace
- **Which-key**: Keymap hints popup

## Text Editing

- **Dial**: Smart increment/decrement
- **Mini.ai**: Enhanced text objects
- **Mini.bracketed**: Bracket-based navigation
- **Mini.surround**: Add/change/delete surroundings
- **Mini.pairs**: Auto-close brackets and quotes
- **Comments**: Smart commenting
- **Inc-rename**: Live rename preview

## UI Enhancements

- **Lualine**: Statusline, winbar, and tabline
- **Noice**: Enhanced command line and notifications
- **Render-markdown**: Live markdown preview
- **Mini.cursorword**: Highlight word under cursor
- **Mini.hipatterns**: Highlight patterns (TODO, hex colors)
- **Mini.trailspace**: Highlight trailing whitespace
- **Mini.indentscope**: Indent guides

## Utilities

- **Database**: Database client
- **Session**: Session management
- **Toolbox**: Custom command palette
- **CSV**: CSV file support

## Auto-installed Tools

- **Language Servers**: clangd, pyright, lua-language-server, marksman
- **Formatters**: black, clang-format, stylua, shfmt, prettier
